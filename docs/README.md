## 📱 アプリ開発関連

### アーキテクチャ
- Clean Architecture + Riverpod
- Firebase バックエンド
- Flutter 3.24.3 + Dart 3.5.3

### 主要機能
- ユーザー認証（Firebase Auth）
- スケジュール管理
- 友達機能
- プッシュ通知
- AdMob広告統合
